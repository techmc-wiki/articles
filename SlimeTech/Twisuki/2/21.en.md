# #21 Mobility

---

In this section, we aim to achieve the goal of _extending "mobility" into structures_ to _making structures mobile_. That is, instead of mounting things onto engines, we'll make functional structures fly directly.

## Basic Mobility Structures

In the previous section, we explained the principles of mobility and provided the basic 10gt flying machine. From the 10gt flying machine, we can extract the mobility structure as follows:

![img.png](img/2.1.1-10gt.png)

The necessary structures are:

- Two opposing pistons, at least one being a sticky piston
- Two observers, each activating one piston
- Slime blocks for connection

To extend a functional structure into a flying machine, simply complete these three necessary structures.

Since functional structures inevitably contain slime blocks (rarely none), prioritize using existing slime blocks. It's best not to introduce new slime blocks.

Next, if the functional structure contains observer-piston structures, prioritize utilizing existing observers. That is, add pistons to positions where observers can activate them. It's best not to introduce new observers.

Finally, place the pistons. Be careful to avoid exceeding the push limit and ensure all structures can be connected and moved.

## Examples

What follows is practice. This section cannot be divided more finely—practice makes perfect. Moreover, the above principles are not mandatory. Some players prefer introducing new slime blocks or observers, which can actually make the entire machine more concise and elegant.
