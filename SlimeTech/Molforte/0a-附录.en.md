## Terminology Reference Table

**Since different Slime Tech players have different interpretations of terminology, to ensure accuracy throughout the chapter, a terminology reference table is provided here**. Additionally, the following terms will also appear in the article as footnotes (not yet completed).

- Flying machine: In a broad sense, flying machine refers to Slime Tech; in a narrow sense, it refers to modules that can move themselves. This text follows the narrow definition.
- Engine: Refers to flying machines, usually flying machines that do not move in 10gt cycles. This chapter follows the narrow definition.
- Piston worm: In a broad sense, refers to flying machines; in a narrow sense, refers to movable flying machines consisting only of piston heads and piston heads directly opposite each other. This chapter follows the narrow definition.
- Self-returning flying machine: A flying machine that automatically returns after encountering an obstacle (such as hitting obsidian)
- Automatic: Refers to a flying machine that can move itself, i.e., moving by stepping on its own feet.
- Stepper: A module that performs limited movement after being activated, such as single-step: moves once when activated once; double-step: moves twice or two blocks when activated once.
- Mounting: Attaching another module to a module via pistons or other means is called mounting. The mounted module is usually not automatic, or the mounted module does not react to the mounting module.
- Docking station: A Slime Tech structure that can dock flying machines and launch flying machines. Can launch flying machines after stepping, or with 0-step (i.e., the flying machine does not move)
- Departure station: The docking station where machines like world eaters start by default.
- Return station: The other end of the default departure station is the return station. Can also serve as a docking station, or may not be able to dock flying machines.
- Extension: In this text, refers only to extending repeating units such as world eaters, mining machines, tunnel bores, etc., and can also be equivalent to stacking.
- Piston chain: Usually a chain-like device consisting of sticky pistons alternating with slime blocks, typically used as a delay or block conveyor belt.
