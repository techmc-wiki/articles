## Brief Introduction to Litematica

By default, the hotkey for the Litematica main menu is M. Press it to open the main menu.


In the configuration menu, you can change hotkeys and toggle various Litematica features.
When holding the Litematica tool in your main hand or offhand, the Litematica menu bar will appear in the lower left corner.

![](https://www.xhbsh.top/img/mod/litematica/img_1.png)

Use ctrl+scroll wheel to switch modes.

### Mode 1 -- Area Selection

There are two modes for area selection, which can be switched in the main menu. Simple mode is recommended for beginners, and in the area editor, set the corner mode to corners.

#### Simple Mode

In simple mode, there is only one selection area. Use left-click and right-click to determine the two projection origin points to confirm the selection area.
Use middle-click on a corner (or other positions in the selection) to select the corner (or selection). After selecting, hold `alt` and scroll the mouse wheel to move the corner (or selection).

![](https://www.xhbsh.top/img/mod/litematica/img_2.png)

In the selection editor, you can change the name of the selection or save the schematic.

![](https://www.xhbsh.top/img/mod/litematica/img4.png)

#### Normal Mode

For large machines, using simple mode for saving can cause various inconveniences during construction. Normal mode solves this problem well.
In normal mode, the selection area is not automatically created and needs to be manually created by the player. The steps are as follows:

1. Open `Litematica Main Menu - Selection Manager - New Selection`

![](https://www.xhbsh.top/img/mod/litematica/img_4.png)

2. If you need to create a sub-selection, select "New Sub-region" in `Litematica Main Menu - Area Editor`. You can also modify the `Add Selection` hotkey in Configuration Menu - Hotkeys for quick and convenient addition of a selection.
Use middle-click to select a sub-selection corner (or sub-selection).
The final result is shown in the image ![](https://www.xhbsh.top/img/mod/litematica/img_5.png)
The steps to save the schematic are the same as in simple mode.

### Mode 2 Schematic Placement & Mode 5 Schematic Paste

Import external schematics (skip if none).
It is recommended to enable the custom schematic directory in the projection settings, so that all versions can share one save location for schematics.

![](https://www.xhbsh.top/img/mod/litematica/img_6.png)

Then place the downloaded schematics from the internet into this folder.

1. Load the schematic in-game

`Litematica Main Menu - Load Schematics` Select the desired schematic and click "Load Schematic" in the lower left corner.

1. Adjust the position of the schematic and rotate the schematic

Use middle-click to select the schematic, alt+mouse wheel to move the position.

`Litematica Main Menu - Configure Schematic` Select the desired schematic and click the configure button to enter the configuration interface.

![](https://www.xhbsh.top/img/mod/litematica/img_8.png)

On the right side of the configuration interface, you can adjust the position of the projection origin and rotate/mirror the schematic.

Note! Unless stated by the author, it is not recommended to rotate/mirror when building machines, as this is very likely to cause the machine to malfunction.
